# pmg-tracing-interface

O PMG Tracing é uma ferramenta indispensável para administradores de TI e provedores de hospedagem que utilizam o Proxmox Mail Gateway (PMG). Desenvolvido para simplificar e otimizar o gerenciamento e a auditoria de e-mails, ele transforma a complexa tarefa de analisar logs do servidor em uma experiência visual, ágil e totalmente segura.

Com uma interface centralizada, o PMG Tracing permite que equipes de suporte e os próprios clientes finais identifiquem rapidamente o destino de qualquer mensagem, solucionando problemas de entrega em questão de segundos.

Sistema Multi-Usuário: Permite a criação de múltiplos perfis de acesso, ideal para equipes de suporte técnico com diferentes níveis de atribuição. Isolamento por Domínio: Segurança e privacidade garantidas, onde cada usuário ou cliente cadastrado consegue visualizar e acessar estritamente os logs dos seus próprios domínios. Busca Facilitada por Range de Datas: Encontre mensagens antigas ou específicas sem esforço, filtrando os logs por intervalos personalizados de dias ou horas. Rápida Visualização do Status das Mensagens: Identifique instantaneamente se um e-mail foi aceito, rejeitado, colocado em quarentena ou se está na fila de envio, graças a um painel intuitivo. Autenticação em Dois Fatores (2FA): Camada extra de proteção que exige uma validação secundária para o login, blindando os dados de rastreamento contra acessos não autorizados. Integração com Cloudflare Turnstile: Proteção inteligente contra ataques de força bruta e acessos automatizados (bots) na tela de login, sem o incômodo dos CAPTCHAs tradicionais. Agilidade no Suporte ao Cliente: Reduz drasticamente o tempo de resposta do suporte técnico, pois elimina a necessidade de buscar logs manualmente via linha de comando (CLI). Interface Amigável e Responsiva: Design limpo que traduz códigos técnicos de servidores de e-mail em informações claras, acessíveis de qualquer dispositivo.
